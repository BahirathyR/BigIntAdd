## What is this?

In this module used to add Big or Long Integer.

## Installation

npm i bigintegeradd --save

Then...

```
import {bigintegeradd} from 'bigintegeradd'
 
 var a="234567899909"
 var b="678905432235"

 BigIntAdd(a,b)

 ```

  ## Add module to import project package.json
  {
      "type":"module"
  }

  ## options
  var a="234567899909"
  var b="678905432235"
 
 ## or
  var a="234567899909"
  var b="67890543223567886"

  Long Integer value only.
