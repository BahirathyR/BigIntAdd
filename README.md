## What is this?

In this module used to add Big or Long Integer.

## Installation

npm i bigintegeradd --save

Then...

```
const {bigintegeradd} = require("bigintegeradd")

var a="34564893479"
var b="47934975474"

if(a.length>=b.length){
    console.log(bigintegeradd(a,b))
}
else
{
    bigintegeradd(b,a)
}

 ```
